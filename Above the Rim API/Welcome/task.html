Description

You and your friends are huge basketball fans. Some of you are varsity players, but as a group of enthusiasts, you edit local online information services in schools. Unfortunately, there are no TV broadcasts at this level of competition, and no major website publishes the final results. To find out what the result is, you must either write to or call your colleagues. You decided to change it and run the API so that each of your teammates can submit the result after the match is over, and others can check it at any time.

Objectives
  1. On the home page, return a greeting with the <h1> header: Welcome to the "Above the Rim" API! The service should return the status code of 200;
  2. Handle the 404 error and return the JSON object with the status code 404. 
    {"success": False, "data": "Wrong address"}
  3. Create the SQLite3 database db.sqlite3 with the table teams and columns:
Column Type Attributes
ID Integer Primary Key
SHORT String(3) Unique, Not null
NAME String(50) Unique, Not null

Examples

Example 1: The response for the home page should be HTML text in <h1> tag.

Welcome to the "Above the Rim" API!

Example 2: Response for a GET to a wrong address should return a JSON object.

{"success": False, "data": "Wrong address"}
